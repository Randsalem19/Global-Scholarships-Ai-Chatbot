## 📌 Project Overview
**ScholarAI** is a state-of-the-art, full-stack conversational assistant designed to help students discover and apply for fully-funded global scholarships. Built with a focus on deep Natural Language Processing (NLP) and robust data retrieval, this project demonstrates the practical integration of Large Language Models (LLMs) into web applications using a **Retrieval-Augmented Generation (RAG)** architecture.

Developed by a researcher with a strong background in Data Science and NLP, the system dynamically bridges the gap between complex scholarship data and user-friendly, intelligent interactions.

## 🚀 Key Features

* **🧠 Intelligent RAG Pipeline:** Context-aware responses generated by querying a dynamic JSON knowledge base of 2026 global scholarships (e.g., Chevening, Fulbright, DAAD) to completely eliminate AI hallucinations.
* **🌍 Bilingual & Native RTL Support:** Seamlessly processes both Arabic and English queries. The UI dynamically adapts text direction (Right-to-Left for Arabic) ensuring a native, highly accessible user experience.
* **🕷️ Automated Data Scraping:** Features a custom-built Python web scraper (`scraper.py`) utilizing `BeautifulSoup4` to automatically extract, clean, and update the internal database with fresh scholarship opportunities from external sources.
* **✨ Premium Glassmorphism UI:** A highly responsive, immersive dark-themed dashboard inspired by top-tier academic institutions, featuring blurred glass effects, interactive quick-action prompts, and real-time typing indicators.

## 🛠️ Technology Stack

**Backend:**
* Python 3.x
* FastAPI & Uvicorn (High-performance API routing)
* Groq API / OpenAI (LLaMA3 / GPT Models)
* BeautifulSoup4 & Requests (Web Scraping & Data Extraction)

**Frontend:**
* HTML5 & CSS3 (Advanced Glassmorphism & CSS Animations)
* Vanilla JavaScript (ES6+ for asynchronous API handling)
